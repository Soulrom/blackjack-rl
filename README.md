## 📖 About

**Playing With Cards** is a terminal-based Blackjack game where you play against a dealer.
Place bets, manage your balance, and try to beat the house — all from your terminal.

Choose between a **standard dealer** (hits until score ≥ 17) or an **AI dealer** trained via
Q-learning reinforcement learning across 5 million simulated games.

## ⚙️ Getting Started

### Prerequisites

```bash
git clone https://github.com/Soulrom/playing-with-cards.git
cd playing-with-cards
```

### Run

```bash
cd src
python main.py
```

## 🤖 AI Dealer

The AI dealer uses a Q-learning agent trained on 5 million episodes.

### Train the model

```bash
cd src
python train.py
```

Training takes a few minutes and saves the Q-table to `models/q_table.json`.
A pre-trained model is not included — run `train.py` before selecting the AI dealer.

### How it works

The agent learns a policy over states `(dealer_score, player_score, has_ace)`.
At each step it chooses to **hit** (1) or **stand** (0) to maximise the expected reward:

```
Q(s, a) ← Q(s, a) + α · (r + γ · max Q(s') − Q(s, a))
```

| Hyperparameter | Value | Role |
|----------------|-------|------|
| α (alpha) | 0.05 | Learning rate |
| γ (gamma) | 0.95 | Future reward discount |
| ε decay | 0.999999 | Exploration → exploitation over ~3M episodes |
| States learned | ~280 | Unique (dealer, player, ace) combinations |

## 🕹️ How to Play

1. Enter your name
2. Choose dealer mode: standard or AI
3. Place a bet (starting balance: $1000)
4. Choose your action each turn:
   - `h` — hit (take another card)
   - `s` — stand (stop and let the dealer play)
5. Try to get closer to **21** than the dealer without going over

### Payouts

| Result | Payout |
|--------|--------|
| Win | +bet |
| Blackjack (21 from 2 cards) | +bet × 1.5 |
| Lose | −bet |
| Draw | bet returned |
